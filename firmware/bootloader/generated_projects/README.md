# Generated MPLAB X projects

Create these projects locally with MPLAB X/MCC:

```text
bootloader_pic32mx534.X/
test_app_pic32mx534.X/
```

Generated Harmony source is intentionally not fabricated in this package. It depends on the installed device family pack, Harmony package versions, MCC version, and XC32 version.

After generation, commit the projects or document a reproducible generation procedure and package manifest. Do not commit compiler caches or private machine paths.
